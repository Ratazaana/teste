Conheça um pouco dos conteúdos abordados:

HTML

HTML (HyperText Markup Language) é a linguagem usada para estruturar e exibir conteúdo na web, como textos, imagens e links.

CSS

CSS (Cascading Style Sheets) é a linguagem usada para estilizar elementos HTML, definindo cores, fontes, espaçamentos e o layout das páginas web.

JavaScript

JavaScript é uma linguagem de programação usada para tornar as páginas web interativas, permitindo ações dinâmicas como cliques, animações e atualizações em tempo real.
